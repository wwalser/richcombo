Examples

No Styles

The first example is very simple with no additional styling. This should give a good idea of the basic approach of the richcombo plugin.
Calling .richcombo on a select element will replace that select with markup that when styled correctly allows for a much richer combo box than offered natively by the browser.

Basic Styles

The second example is simply the first example with styles applied to the richcombo. The default behavior of the plugin gives enough markup to style a rich combo box to fit the asthetic of a surrounding page.

Drop Down with Icons

The third example has styles applied and the background images necessary to make the richcombo appear as a drop down list of iconed selection options.
